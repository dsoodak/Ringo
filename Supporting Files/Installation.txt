Install the Arduino IDE (ex: arduino-1.6.5-r2-windows).exe
Install the latest FTDI driver (ex: CDM_v2.12.00_WHQL_Certified.exe)
Settings:
  programmer: Arduino as ISP
  board: Arduino FIO
  port: whichever one popped up when you plugged in the programmer
  serial monitor settings: 57600 baud, and for some of the examples, "add newline"
Make copy of these folders in "libraries" directory (ex: C:\Program Files (x86)\Arduino\libraries)
  RingoWire
  Adafruit_NeoPixel (use this copy or go to their website for the latest version)
